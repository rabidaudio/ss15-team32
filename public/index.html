Drop-in

First add this tag where you want your comments block to go:

<qcomment></qcomment>

Then, add this with your scripts:

<script src="http://ss15-team32.divshot.io/lib/quick-comments.min.js"></script>
<script>
  new QC({
    firebase: new Firebase('https://<YOUR-FIREBASE-SITE>.firebaseio.com')
  });
</script>

Voila! See the docs for more.

Bower

bower install quick-comments
